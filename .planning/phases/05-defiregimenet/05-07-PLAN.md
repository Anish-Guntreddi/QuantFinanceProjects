- label_distribution: dict[str, dict]     # per token, counts per label state (report data section)

Model comparison protocol (the honest evaluation):
- y: stacked per-token make_regime_labels(...) on (date, token) MultiIndex aligned to the
  feature panel; horizon H from config (default 5)
- Classifiers: RegimeCVEvaluator(embargo_size=H, purged_size=H, label_horizon=H).evaluate
- HMM/GMM baselines: causal regime sequence at t is the persistence prediction for the label
  at t (predicting the forward-looking regime class). Score on the same valid_mask rows the
  classifiers were scored on where possible; accuracy via direct compare, log_loss via
  labels_to_probas(seq, 4, eps=1e-3). Note in a code comment: baselines use the full causal
  sequence (no CV needed — no training on labels occurs), classifiers use purged CV; this
  asymmetry favors the baselines if anything, which is the conservative/honest direction.
